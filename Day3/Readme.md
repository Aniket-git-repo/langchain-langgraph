**Pre-Requisites**

Create a .env file with **GEMINI_API_KEY** and put it under the root directory
The script names are provided beside the assignment topic, Edit the file **input_content.py** (in root folder) to provide new inputs to the script. Details mentioned below

**Assignment 1**

Navigate to the directory Day2 and refer to the below script

 Executive Summary : executive_summary.py (Input the PDF document to summarize along with full path. To be used to summarize PDF documents only)
 Client Email Draft : Client_email_draft.py
 Policy Compliance Checker : policy_compliance_checker.py (Edit the variable **policy_text** under input file )
 Meeting Minutes Summarizer : meeting_summarizer.py (Edit the variable **meeting_transcript** under input file )
 Market Analysis Brief : market_research_analyzer.py

**Assignment 2**

Navigate to the directory Day3 and refer to the below script

 Prompt Quality Scoring : Prompt_quality_scoring.py (Edit the variable **sample_prompt** under input file )

**Assignment 3**

Navigate to the directory Day4 and refer to the below script
 
 Customer Support Agent : Customer_support_agent.py (Edit the variable **email_details** for input e-mail and **knowledge_base_doc_string** for the knowledge base document )
